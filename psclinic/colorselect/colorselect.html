An introduction to the Select by color range

Files for the lesson are available at this link: Color Select Lesson Files (Save this file)

  1. Select -> Color Range
  2. Use the dropper tools in the Color Range pop-up menu to choose the color that will be selected. The Dropper-Minus tool will remove colors from the selection.
  3. Adjust the fuziness to increase/decrease the selection.
  4. Focus on getting a clean border in your selection.
  5. Hit "OK". You will most likely need to fine-tune the selection in the Quick Mask mode (see previous excercise).
  6. Save your selection by choosing "Save Selection" in the Select menu.
  7. Your selection will be saved as an Alpha-Channel, which you can see under the Channels tab.
