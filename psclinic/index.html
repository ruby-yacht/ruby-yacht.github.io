Photoshop Clinic

The purpose of this clinic is to make sure you have the basic skills to do the Mashup Project.

Part 1: Quick Mask - an easy selection method based on drawing in the picture

Part 2: Select by Color - best used when what you are selecting cannot be drawn, but is basically one color

Part 3: Blending - creating illusion of depth and integrating things within an image

Place finished files in appropriate drop box.
